# ACS Landing Page - Next.js App Router Project

A modern **Next.js App Router E-commerce Landing Page** with integrated **Admin Dashboard**.  
Manage products, orders, settings, theme, and all content without touching code.

---

## 🚀 Features

- Hero section management (offer date, badges, titles, descriptions, images)
- Product management (add, edit, delete)
- Order management (track and update orders)
- Settings control (logo, name, details, social links, theme colors)
- FAQ, Feedback, Risk-Free Order sections — all editable
- Responsive design with TailwindCSS
- Smooth animations with Framer Motion
- Authentication with Clerk
- Rich text editing with Tiptap
- Database powered by Mongoose

---

## 🛠 Tech Stack

- Next.js 15 (App Router)
- TypeScript + Zod
- Mongoose
- TailwindCSS + Framer Motion
- Clerk Authentication
- Tiptap Editor
- Radix UI Components
- UploadThing, Chart.js, Lucide React Icons
